List Requirement :

- Spring Boot v2
- Java v11
- Database PostgreSQL

How To Run Apps : 

- Just run through MoviesApplication

List APIs :

2. Movie search (/movie-search) :
	Search movie title using keyword

3. Movie Detail (/movie-detail) :
	Get movie detail
	
4. Movie Comment (/movie-comment) :
	To post comment on movie
